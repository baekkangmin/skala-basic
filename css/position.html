fixed (화면 고정)
1) position: static (기본값)
top/left를 줘도 이동하지 않음. 문서 흐름(레이아웃) 그대로.
일반 박스
static
(top/left 무시)
일반 박스
2) position: relative
원래 위치 기준으로 이동. 이동해도 원래 자리의 공간은 그대로 남아있음.
일반 박스
relative
(top/left 적용)
일반 박스
3) position: absolute
문서 흐름에서 빠지고, 가장 가까운 position 조상(.abs-parent)을 기준으로 배치.
이 점선 박스가 absolute의 기준(부모: position: relative)
일반 박스
일반 박스
일반 박스
absolute
(부모 기준)
4) position: sticky
스크롤하다가 top 위치에 도달하면 “붙어서” 따라다님. (top 필수)
sticky (top:0)
내부 스크롤을 내려보면 sticky가 위에 달라붙어 유지됨.

sticky는 "스크롤 컨테이너" 안에서만 동작해.
(여기서는 .sticky-wrap가 overflow:auto라 내부 스크롤 기준)
